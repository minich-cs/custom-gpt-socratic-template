# Socratic Custom GPT Template

A forkable Custom GPT configuration that uses the Socratic method to support teacher-guided, inquiry-based learning.

This repository provides a structured set of prompt, configuration, and documentation files that educators and students can study, adapt, and extend when building their own Custom GPTs.

## Purpose

This project is designed to:
- Encourage questioning over answer-giving
- Support inquiry-based and reflective learning
- Keep teachers in the instructional loop
- Model responsible, transparent use of AI in education

It is not intended to replace instruction, assessment, or human judgment.

## Who This Is For

- Secondary and post-secondary educators
- Students learning prompt design and AI literacy
- Anyone interested in Socratic, question-driven learning systems

## How to Use

1. Fork this repository.
2. Review the files in the `config/` and `docs/` directories.
3. Modify the prompts and configuration to match your subject and teaching goals.
4. Paste the adapted content into a Custom GPT configuration in ChatGPT.
5. Test, reflect, and iterate.

## Repository Structure

- `config/` — Core prompt and configuration files
- `docs/` — Guidance on customization and classroom use
- `examples/` — Sample subject-specific adaptations (optional)
